Öğrenme Yol Haritası

Başlangıçtan ileri seviyeye kapsamlı eğitim planı

1

Temel Bilgiler (0-3 Ay)

Moleküler Biyoloji Temelleri

  • DNA Yapısı: Nükleotidler (A, T, G, C), baz çiftleri, çift sarmal yapı, antiparalel zincirler
  • RNA: mRNA (mesajcı), tRNA (transfer), rRNA (ribozomal) ve işlevleri
  • Proteinler: 20 amino asit, primer/sekonder/tersiyer yapı, protein fonksiyonları
  • Dogma Merkezi: DNA replikasyonu, transkripsiyon (DNA→RNA), translasyon (RNA→Protein)
  • Genler ve Genomlar: Gen yapısı, ekson, intron, promoter, enhancer, genom organizasyonu
Kaynaklar: Khan Academy Biology Nature Scitable

Programlama Temelleri - Python

  • Temel Sözdizimi: Değişkenler, veri tipleri (int, float, string, bool), operatörler
  • Kontrol Yapıları: if-else koşulları, for ve while döngüleri, break/continue
  • Veri Yapıları: Listeler, sözlükler (dictionary), tuple, set ve kullanımları
  • Fonksiyonlar: Fonksiyon tanımlama, parametreler, return değerleri, lambda fonksiyonları
  • Dosya İşlemleri: Okuma/yazma, CSV, FASTA, FASTQ formatları ile çalışma
  • Kütüphaneler: NumPy (sayısal hesaplamalar), Pandas (veri analizi), Matplotlib (görselleştirme)
Kaynaklar: Python.org Codecademy Real Python

Linux ve Komut Satırı

  • Temel Komutlar: ls (listeleme), cd (dizin değiştirme), pwd, mkdir, rm, cp, mv
  • Dosya İşlemleri: cat, less, head, tail, grep (arama), sed, awk (metin işleme)
  • Pipe ve Yönlendirme: | (pipe), > (yönlendirme), >> (ekleme), < (girdi)
  • Bash Scripting: Değişkenler, döngüler, koşullar, fonksiyonlar, otomatizasyon
  • Paket Yönetimi: apt (Ubuntu/Debian), conda (bilimsel paketler), pip (Python paketleri)
Kaynaklar: Linux Journey Command Line Course
2

Biyoinformatik Temelleri (3-6 Ay)

Dizi Hizalama (Sequence Alignment)

  • Global Hizalama: Needleman-Wunsch algoritması, dinamik programlama, traceback
  • Lokal Hizalama: Smith-Waterman algoritması, en iyi lokal eşleşme bulma
  • Puanlama Matrisleri: BLOSUM (protein), PAM matrisleri, gap penaltıları
  • BLAST: Basic Local Alignment Search Tool, hızlı dizi araması, E-value
  • Çoklu Dizi Hizalama: ClustalW, MUSCLE, MAFFT, progressive alignment
  • Pratik: Biopython ile dizi hizalama, pairwise2 modülü

Biyolojik Veritabanları

  • NCBI: GenBank (nükleotid dizileri), RefSeq (referans diziler), PubMed (makaleler)
  • UniProt: Protein dizileri, fonksiyonel anotasyonlar, Swiss-Prot/TrEMBL
  • PDB: Protein Data Bank, 3D protein yapıları, kristalografi verileri
  • Ensembl: Genom tarayıcısı, gen anotasyonları, karşılaştırmalı genomik
  • KEGG: Kyoto Encyclopedia, metabolik yolaklar, genom haritaları
  • API Kullanımı: Entrez Programming Utilities, REST API'ler, otomatik veri çekme

Dizi Analizi ve Motif Bulma

  • ORF Bulma: Açık okuma çerçeveleri (Open Reading Frames), start/stop kodonları
  • Motif Arama: Düzenli ifadeler (regex), PROSITE patterns, konservatif bölgeler
  • Filogenetik Analiz: Evrimsel ağaçlar, UPGMA, neighbor-joining, maksimum likelihood
  • Sekonder Yapı Tahmini: RNA katlama, protein sekonder yapı (α-helix, β-sheet)
  • Domain Analizi: Pfam (protein aileleri), InterPro, domain mimarisi
3

Yeni Nesil Dizileme - NGS (6-9 Ay)

NGS Teknolojileri

  • Illumina: Sentez ile dizileme (sequencing by synthesis), kısa okumalar (150-300 bp)
  • PacBio: Tek molekül gerçek zamanlı dizileme (SMRT), uzun okumalar (10-30 kb)
  • Oxford Nanopore: Nanopore dizileme, ultra-uzun okumalar (100+ kb)
  • Veri Formatları: FASTQ (ham okumalar), SAM/BAM (hizalanmış okumalar), VCF (varyantlar)
  • Kalite Kontrol: FastQC (kalite raporları), MultiQC (toplu raporlama), Phred skorları

RNA-Seq Analizi

  • Read Hizalama: STAR (splice-aware), HISAT2, Bowtie2, referans genom hizalama
  • Transkript Birleştirme: StringTie, Cufflinks, transkript keşfi ve kantifikasyonu
  • Diferansiyel İfade: DESeq2, edgeR, limma-voom, istatistiksel testler
  • Fonksiyonel Analiz: GO (Gene Ontology), KEGG pathway zenginleştirme, GSEA
  • Görselleştirme: Heatmap, volcano plot, MA plot, PCA, gene expression profilleri

Varyant Analizi (Variant Calling)

  • Read Preprocessing: Trimmomatic (adapter kesme), Cutadapt, kalite filtreleme
  • Genom Hizalama: BWA-MEM, Bowtie2, referans genom indeksleme
  • Varyant Çağırma: GATK HaplotypeCaller, FreeBayes, SAMtools mpileup, SNP/Indel
  • Anotasyon: SnpEff, ANNOVAR, VEP (Variant Effect Predictor), fonksiyonel etki
  • Filtreleme: Kalite skorları (QUAL), derinlik (DP), popülasyon frekansları (AF)
4

İleri Düzey Analiz (9-12 Ay)

Makine Öğrenmesi ve AI

  • Denetimli Öğrenme: Sınıflandırma (SVM, Random Forest), regresyon, model eğitimi
  • Denetimsiz Öğrenme: Kümeleme (k-means, hierarchical), PCA, t-SNE, UMAP
  • Özellik Seçimi: Feature engineering, özellik önem skorları, boyut indirgeme
  • Model Değerlendirme: Cross-validation, confusion matrix, ROC-AUC, precision/recall
  • Derin Öğrenme: CNN (konvolüsyonel), RNN (rekürrent), LSTM, transformer modelleri
  • Kütüphaneler: scikit-learn, TensorFlow, PyTorch, Keras
  • Uygulamalar: Protein yapı tahmini, gen ifade tahmini, varyant patogenite

Tek Hücre Analizi (Single-Cell)

  • scRNA-seq: Tek hücre RNA dizileme, hücre heterojenitesi, 10x Genomics
  • Preprocessing: Kalite kontrol, normalizasyon, batch effect düzeltme
  • Kümeleme: Hücre tipi tanımlama, marker gen bulma, cell type annotation
  • Trajectory Analizi: Pseudotime analizi, hücre farklılaşma yolakları, Monocle
  • Araçlar: Seurat (R), Scanpy (Python), Cell Ranger (preprocessing)
  • Görselleştirme: UMAP, t-SNE, violin plots, feature plots, dot plots

Sistem Biyolojisi

  • Ağ Analizi: Protein-protein etkileşim (PPI) ağları, network topology
  • Yolak Analizi: Metabolik yolaklar, sinyal yolakları, pathway enrichment
  • Entegratif Analiz: Multi-omics veri entegrasyonu (genomik, transkriptomik, proteomik)
  • Matematiksel Modelleme: ODE (ordinary differential equations), stokastik modeller
  • Araçlar: Cytoscape (ağ görselleştirme), Reactome, STRING database
5

Uzmanlık Alanları (12+ Ay)

Yapısal Biyoinformatik

  • Protein Yapı Tahmini: AlphaFold2, RoseTTAFold, homology modeling
  • Moleküler Dinamik: GROMACS, AMBER, NAMD, MD simülasyonları
  • Docking: Protein-ligand etkileşimleri, AutoDock, Vina, binding affinity
  • Görselleştirme: PyMOL, Chimera, VMD, moleküler grafik
  • Yapı Analizi: RMSD, RMSF, Ramachandran plot, yapı kalitesi

İlaç Keşfi ve Tasarımı

  • Sanal Tarama: Virtual screening, ligand-bazlı ve yapı-bazlı yaklaşımlar
  • QSAR: Quantitative Structure-Activity Relationship, aktivite tahmini
  • Farmakofor Modelleme: Moleküler özellikler, 3D farmakofor
  • ADMET Tahmini: Absorption, Distribution, Metabolism, Excretion, Toxicity
  • Lead Optimization: İlaç adayı optimizasyonu, Lipinski kuralları

Klinik Biyoinformatik

  • Kanser Genomik: Somatik mutasyonlar, driver/passenger genler, tumor mutasyon yükü
  • Kişiselleştirilmiş Tıp: Farmakogenomik, ilaç yanıt tahmini, precision medicine
  • Nadir Hastalıklar: Varyant yorumlama, Mendelian hastalıklar, trio analizi
  • Klinik Raporlama: ACMG kılavuzları, varyant sınıflandırma (pathogenic/benign)
  • Etik ve Gizlilik: HIPAA, GDPR, veri güvenliği, informed consent
